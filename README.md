# SnapFrame

SnapFrame 是一个固定比例 Mac 截图工具。当前默认取景比例为 `3:4`。

## 功能

- 全屏半透明取景层
- 固定 `3:4` 截图比例
- 松手后保存 PNG 到桌面
- 自动复制 PNG 到剪贴板
- 支持 `Esc` 退出
- 使用 ScreenCaptureKit，支持 Retina 清晰截图

用法：

1. 优先双击 `SnapFrame.app`
2. 如果 `.app` 被系统拦截，右键点击它，选「打开」
3. 也可以双击 `start-native.command` 作为备用入口
4. 屏幕会出现半透明取景层
5. 鼠标拖动选择 `3:4` 区域
6. 松手后自动保存到桌面，并复制 PNG 到剪贴板
7. 按 `Esc` 取消

第一次运行或第一次截图时，macOS 可能要求给 SnapFrame、Terminal 或 Python「屏幕录制」权限。允许后如未立即生效，退出后重新运行。

当前版本按单个显示器取景。多显示器时，在哪个显示器上拖框，就截哪个显示器。

备用命令行用法：

1. 双击 `start-native.command`
2. 屏幕会出现半透明取景层
3. 鼠标拖动选择 `3:4` 区域
4. 松手后自动保存到桌面，并复制 PNG 到剪贴板
5. 按 `Esc` 取消
